## 📘 Descripción general del sistema
Este proyecto corresponde a la actividad sumativa 2, aca abordaremos el caso de Llanquihue-Tour
creando un sistema de reservas dentro de la empresa, automatizando la obtencion de datos y filtrado de datos, en este caso
las reservas de los tours con todas sus caracteristicas.

## 🧱 Estructura general del proyecto

```plaintext
📁 src/
├── ui/         # Clase principal con el método main
├── model/       # Clase de dominio y clase de composicion (Reserva, Tour)
├── util/     #clase de gestion de datos (LectorDatos)
├── Data/  #clase de servicio (ReservaService) 

````

## ⚙️ Instrucciones para clonar y ejecutar el proyecto

1. Clona el repositorio desde GitHub:

```bash
https://github.com/ariel-malo/LlanquihueTour.git
```

2. Abre el proyecto en IntelliJ IDEA.
3. Crea un objeto en la clase Main.java desde el paquete app
4. Ejecuta el archivo `Main.java` desde el paquete `ui`.
